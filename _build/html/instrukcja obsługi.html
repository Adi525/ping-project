Instrukcja obsługi¶

Aby skorzystać z programu należy pobrać z repozytorium GitHub plik ping.exe, uruchomić go a w razie problemów wyłączyć program antywirusowy. Po uruchomieniu należy wpisać nazwę domeny i liczbę wysyłanych do niej sygnałów oraz opcjonalnie wybrać opcję zapisu do pliku i wpisać ścieżkę gdzie plik ma zostać zapisany. Po uzupełnieniu danych należy nacisnąć przycisk “wykonaj” i poczekać chwilę na otrzymanie odpowiedzi od domeny. Program wypisuje adres ip domeny, oraz wartości minimalnego, maksymalnego i średniego opóźnienia.
